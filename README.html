Get Started

Our framework streamlines the development process by providing a unified environment for both front-end and back-end development using React and Node.js architecture. With our solution, developers can seamlessly work within a single environment, eliminating the need to switch between separate front-end and back-end setups. This approach enhances productivity, simplifies collaboration, and accelerates the development of web applications.

Create a new project 

npnx create-amacle-app your_project_name

SetUp your .env file


    PORT=3500
    DB_HOST=localhost
    DB_USER=root
    DB_PASSWORD=
    DB_DATABASE=test
    ALLOW_ORIGIN=*
  • PORT: Specifies that the port on which your api will run
  • DB_USER: Specifies the User of your SQl DB
  • DB_PASSWORD: Specifies the password of your SQl DB
  • DB_DATABASE: Specifies the DATABASE NAME
  • ALLOW_ORIGIN: Specifies that the allowd origin of your apis make sure to change it

    • To Run the Project in Developer mode use

    npm run serve

    You will be redirected to http://localhost:your-port/
